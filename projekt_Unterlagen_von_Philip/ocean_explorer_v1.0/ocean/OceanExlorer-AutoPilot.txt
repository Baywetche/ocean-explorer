Schiff-Richtungen: Norden, Süden und Westen

start-Autopilot aus belibiegen Positionen:

- fahre auf sector (0, y):
	- setze Schiff-Richtung auf Westen
	- fahre immer in Schiff-Richtung
	- wenn Hindernis: umfahren()
		
- fahre auf sector (0, 0)
	- setze Schiff-Richtung auf Süden
	- fahre immer in Schiff-Richtung
	- wenn Hindernis: umfahren()
	- setze Schiff-Richtung auf Norden

- fahre gerade aus bis zum y = 98:
	- wenn y = 98, fahre auf (x + 1, y + 1).
	- wenn y = 99, fahre auf (x - 1, y).
	- wenn y = 99, fahre auf (x + 1, y - 1).
	- dann fahre nach Süden
	- setze Schiff-Richtung auf Süden
	
- umfahren:
	- fahre ein Schritt rückwärts zum letzten SchiffSector.
	- Halte dich aber auf die Hand-an-der-wand-Regel
	- ist Schiff-Richtung nach Norden orientiert, fahre das Hindernis von rechts um. 
	- ist Schiff-Richtung nach Süden orientiert, fahre das Hindernis von links um.
	
	- ist Schiff-Richtung nach Westen orientiert und y = 0, fahre das Hindernis von rechts um. 
	- ist Schiff-Richtung nach Westen orientiert und y = 99, fahre das Hindernis von rechts links um. 













ist Schiff-sector (x = 99, y = 0), stopp-Autopilot

















start-Autopilot aus belibiegen Position:

- fahre auf sector (0, y): 1*
	- fahre immer nach Westen
	- wenn blockiert:
		- fahre ein Schritt rückwärts zum letzten SchiffSector.
		- y = 0, fahre auf (x - 1, y + 1)
		- y != 0, fahre auf (x - 1, y)
	- wiederhole oberen Schritte solange, bis die Richtung (0, y).
		
- fahre auf sector (0, 0)
	- y != 0, fahre nach Süden
	- wenn Hindernis:
		- umfahren
	
- richte die Schiff-Richtung nach Norden

- fahre gerade aus bis zum y = 98:
	- wenn y = 98, fahre auf (x + 1, y + 1).
	- wenn y = 99, fahre auf (x - 1, y).
	- wenn y = 99, fahre auf (x + 1, y - 1).
	- dann fahre nach Süden
	- richte die Schiff-Richtung nach Süden.
	
- umfahren:
	- fahre ein Schritt rückwärts zum letzten SchiffSector.
	- Halte dich aber auf die Hand-an-der-wand-Regel
	- ist Schiff-Richtung nach Norden orientiert, fahre das Hindernis von rechts um. 
	- ist Schiff-Richtung nach Süden orientiert, fahre das Hindernis von links um.



ist Schiff-sector (x = 99, y = 0), stopp-Autopilot
